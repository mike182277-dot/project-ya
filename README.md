## Как опубликовать на GitHub Pages
1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из этой папки.
3. Убедитесь, что главная страница называется `index.html`.
4. Откройте `Settings` → `Pages`.
5. В разделе Build and deployment выберите:
   - Source: `Deploy from a branch`
   - Branch: `main` (или `master`)
   - Folder: `/root`
6. Сохраните настройки и дождитесь публикации.
