# Проект Я — иммерсивный интенсив

Лендинг авторского офлайн-интенсива Олеси Славянской и Михаила Качалкина для предпринимателей, собственников и лидеров, которые хотят пересобрать себя как систему: вернуть ясность, энергию и осознанный вектор движения.

«Проект Я» — это 4-дневный иммерсивный интенсив в Геленджике на группу из 5 человек. Участники проходят путь от анализа себя как проекта — через энергоаудит и лидерский опыт — к сборке персональной карты «Проект Я» и подключению ИИ-сопилота для дальнейшего сопровождения.

Формат: структурная работа с собой на стыке стратегической фасилитации, групповой динамики и иммерсивного проживания.
